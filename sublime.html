Package Control

The best plugin for Sublime Text 2 is the Package Control plugin [link]. Package Control lists available plugins and also takes care of installing them, meaning plugin installation is as simple as typing part of a package name (to see what’s available) and choosing from a list. [example]

Zen Coding

The Zen Coding plugin is installed by default in Sublime. This plugin makes creating HTML vastly quicker and easier - instead of typing out each tag and nesting them, you type a snippet version and press ‘Tab’ to expand it into HTML. For example, to create a list with an id of ‘nav’ containing three links, the Zen snippet would be: 

ul#nav>li*3>a
This expands to:

<ul id="nav">
    <li><a href=""></a></li>
    <li><a href=""></a></li>
    <li><a href=""></a></li>
</ul>

    &lt;li&gt;&lt;a href=""&gt;&lt;/a&gt;&lt;/li&gt;
&lt;/ul&gt;

Tag

Tag is one of my favourite plugins when I’m working with HTML - not only does it highlight tags which are missing closing tags, but it also inserts the correct closing tag when you type </ (the start of a closing tag).
